# 📝 React To-Do List Application visit now - https://sandeepgudme.github.io/To-do-list

## 📌 Project Overview

This is a simple To-Do List application built using **React (Vite)**.
The app allows users to manage daily tasks efficiently by adding, editing, deleting, and marking tasks as completed.

---

## 🚀 Features

* ➕ Add new tasks
* ✅ Mark tasks as completed
* ✏️ Edit existing tasks
* ❌ Delete tasks
* 🔄 Dynamic rendering using React state
* 🎨 Basic styling for better user experience

---

## 🛠️ Technologies Used

* React (Functional Components)
* Vite
* JavaScript (ES6)
* CSS

## 🌐 How to Use

1. Enter a task in the input field
2. Click **Add** to add the task
3. Use:

   * ✔ checkbox to mark complete
   * ✏ Edit button to update task
   * ❌ Delete button to remove task

---

## 📌 Notes

* Each task has a unique ID.
* State is managed in the `App` component.
* Props are used to pass data between components.
